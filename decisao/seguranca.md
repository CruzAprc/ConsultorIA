# Regras de Segurança

## 1. Medicações e Interações

| Medicamento | Interação | Ajuste necessário |
|-------------|-----------|-------------------|
| Metformina | Pode causar deficiência B12 | Considerar suplementação B12 |
| Levotiroxina | Tomar em jejum, 30-60min antes de comer | Primeira refeição no mínimo 30min após |
| Corticoides (prednisona) | Aumenta apetite, retém líquido, catabolismo | Proteína mais alta (2.0+ g/kg), monitorar peso |
| Anticoagulantes (warfarina) | Interação com vitamina K | Manter consumo constante de folhas verdes, NÃO suplementar ômega 3 |
| Estatinas | Podem causar dor muscular | Monitorar mialgia, evitar treino excessivo se sintomático |
| Antidepressivos (ISRS) | Podem alterar peso | Monitorar peso, ajustar kcal conforme evolução |
| Insulina | Hipoglicemia com exercício | ENCAMINHAR ENDOCRINOLOGISTA. Não ajustar dieta sem supervisão |
| Hormônio do crescimento | Altera composição corporal | Ajustar macros com mais frequência |
| Anticonvulsivantes | Alguns aumentam apetite | Monitorar peso |
| Isotretinoína (Roacutan) | Função hepática | Evitar suplementos hepatotóxicos, moderar gordura saturada |

## 2. Condições Crônicas → Restrições

### Diabetes Tipo 2
- Carboidrato ≤ 3.0 g/kg
- Priorizar carboidratos de baixo índice glicêmico
- Evitar: açúcar adicionado, sucos, mel, doces
- Distribuir carbs uniformemente nas refeições
- Incluir fibras em todas as refeições

### Hipotireoidismo
- TDEE real pode ser 10-20% menor que calculado
- Aplicar -10% no TDEE
- Evitar dietas muito restritivas (< 1200 kcal pode piorar)
- Verificar se usa levotiroxina corretamente

### Hipertensão
- Sódio < 2000mg/dia
- Evitar: embutidos, enlatados, queijos curados, temperos prontos
- NÃO recomendar cafeína pré-treino
- Evitar exercícios isométricos prolongados (prancha > 60s)
- Preferir aeróbico + musculação moderada

### SOP (Síndrome dos Ovários Policísticos)
- Carboidrato ≤ 2.5 g/kg
- Priorizar baixo IG
- Gordura mínima 0.7 g/kg (saúde hormonal)
- Considerar incluir: canela, omega 3, magnésio

### Doença Renal
- **NÃO PRESCREVER sem aval médico**
- Proteína ≤ 0.8 g/kg (se clearance < 60)
- ZERO creatina
- Potássio e fósforo limitados
- Encaminhar nefrologista

### Doença Celíaca
- ZERO glúten: trigo, cevada, centeio, aveia contaminada
- Verificar tags "sem_gluten" no banco de alimentos
- Cuidado com: aveia (só usar certificada sem glúten), molhos prontos

### Intolerância à Lactose
- Remover: leite, iogurte regular, queijos frescos, whey concentrado
- Permitido: whey isolado, queijos curados (parmesão, cheddar), iogurte sem lactose
- Substituir por: bebida vegetal, iogurte vegetal, whey isolado/vegetal

## 3. Lesões → Adaptações Obrigatórias

> Consultar `treinos/adaptacoes-lesao.json` para lista detalhada

### Regras gerais:
1. SEMPRE respeitar a lesão reportada
2. Se em dúvida, REMOVER o exercício (não arriscar)
3. Sugerir alternativa com máquina ou amplitude reduzida
4. Incluir nota: "Em caso de dor, interromper imediatamente e consultar fisioterapeuta"
5. Reduzir carga em 30-50% para grupo muscular afetado
6. Priorizar exercícios com amplitude controlada (máquinas)

### Lesões mais comuns (base de dados):
1. Joelho (17 casos) → adaptar MMII
2. Ombro (25 casos total) → adaptar peito, ombro, tríceps
3. Lombar (14 casos) → adaptar costas, posterior
4. Cotovelo/epicondilite (8 casos) → adaptar bíceps, tríceps
5. Calcanhar/pé (10 casos) → adaptar panturrilha, evitar impacto
6. Tendão rotuliano (5 casos) → adaptar extensora, agachamento

## 4. Restrições Alimentares → Checklist

### Vegetariano
- ✓ Ovos e laticínios permitidos
- ✗ Carne, frango, peixe
- Proteínas: ovos, queijos, iogurte, tofu, PTS, grão-de-bico, lentilha
- Considerar: whey, creatina (ambos compatíveis)

### Vegano
- ✗ Zero origem animal
- Proteínas: tofu, PTS, grão-de-bico, lentilha, ervilha
- Suplementos: whey vegetal (ervilha/soja), B12 obrigatório, creatina OK
- Atenção: ferro, zinco, ômega 3 (suplementar DHA de alga)

### Sem Lactose
- ✗ Leite, iogurte regular, queijos frescos, whey concentrado
- ✓ Whey isolado, queijos curados, versões sem lactose
- Substituir laticínios por versões sem lactose ou vegetais

### Sem Glúten
- ✗ Pão, macarrão, aveia contaminada, granola com trigo
- ✓ Arroz, batata, tapioca, cuscuz, quinoa, aveia sem glúten
- Verificar tag "sem_gluten" em todos os alimentos
